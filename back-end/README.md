# Django Rest Framework API

### Technologies Used:

- Django Rest Framework
- Postegres
- Python

### How To Run

install Django to venv:

Windows
```
python -m venv venv

.\venv\Scripts\Activate.ps1 
or
.\venv\Scripts\activate.bat

pip install -r requirements.txt
```

Linux
```
python3 -m venv venv

.venv/bin/activate

pip install -r requirements.txt
```

Execulte the code 
```
python manage.py runserver
```
